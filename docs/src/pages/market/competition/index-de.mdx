# Vergleichbare Dienstleistungen

Vergleichbare Dienstleistungen werden in drei unterschiedlichen Kategorien
eingeordnet.

## Hypothekenempfehlung

Die Berechnung von Hypotheken wird durch mehrere Online Rechner bereitgestellt.
Dabei werden vom Nutzer Angbaben zum Objekt und personen bezogene Daten
abgefragt.

> Dienstleistungen für Hypotheken-Berechnungen geben Empfehlung für mögliche
> Hypotheken die den Rahmen der abgeraften Daten liegen. Dem Nutzer werden
> konkrete Angebote dargestellt jedoch enthalten diese eine Aufforderung einer
> direkten Kontaktaufnahme um Einzelheiten auszutauschen.
